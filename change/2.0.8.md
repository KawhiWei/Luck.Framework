# change 2.0.8 version

## Add
### 1、全面升级目标框架支持
+ 添加对 .NET 9.0 和 .NET 10.0 的支持
+ 所有项目现在支持多目标框架：net6.0;net7.0;net8.0;net9.0;net10.0

### 2、Luck.Framework
+ 添加 .NET 9.0 和 .NET 10.0 的 Microsoft.Extensions.* 包引用

```xml
<ItemGroup Condition="'$(TargetFramework)' == 'net9.0'">
  <PackageReference Include="Microsoft.Extensions.Configuration.Abstractions" Version="9.0.0" />
  <PackageReference Include="Microsoft.Extensions.DependencyInjection" Version="9.0.0" />
  ...
</ItemGroup>
<ItemGroup Condition="'$(TargetFramework)' == 'net10.0'">
  <PackageReference Include="Microsoft.Extensions.Configuration.Abstractions" Version="10.0.0-preview.1.25080.5" />
  <PackageReference Include="Microsoft.Extensions.DependencyInjection" Version="10.0.0-preview.1.25080.5" />
  ...
</ItemGroup>
```
